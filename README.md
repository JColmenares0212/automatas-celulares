# Autómatas Celulares

Repositorio complementario a la monografía presentada para el examen final de **Matemáticas Discretas II** (Universidad Nacional de Colombia).  
Aquí se implementan y documentan distintos autómatas celulares unidimensionales con el propósito de ilustrar conceptos de **dinámica discreta**, **computación clásica**, **operaciones booleanas** y **sistemas dinámicos definidos por recurrencias**, todos ellos temas centrales del curso.

Este repositorio busca servir como apoyo visual, computacional y experimental al análisis desarrollado en la monografía, permitiendo explorar los patrones emergentes, la sensibilidad a condiciones iniciales y el comportamiento caótico o complejo de reglas elementales.
